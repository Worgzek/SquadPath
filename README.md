# SquadPath MVP Specification

## MVP Goal

Validate whether users consistently progress through structured career sprints with accountability-based squads.

---

# Included Features

## 1. Path Analyzer
A quiz-based onboarding system that suggests suitable career paths.

### Initial Paths
- Data Engineering
- Data Science
- Frontend Development

### Inputs
- interests
- working style
- learning preferences
- commitment level

### Output
- suggested path
- reasoning
- recommended roadmap

---

## 2. Roadmap System

Each path contains:
- milestones
- sprints
- deliverables
- estimated workload

Roadmaps are manually curated and inspired by existing industry roadmaps.

---

## 3. Sprint Execution System

Each sprint includes:
- learning objectives
- tasks
- deliverables
- completion tracking

---

## 4. Squad Accountability

Users are grouped into small squads based on:
- path
- commitment level
- progression stage

Features:
- progress visibility
- weekly check-ins
- deliverable sharing

---

# NOT Included

The following are intentionally excluded from MVP:
- AI mentors
- real-time chat
- recruiter platform
- project marketplace
- mobile app
- adaptive AI curriculum
- recommendation engine
- social feed

---

# Success Metrics

## Primary Metrics
- week 2 retention
- sprint completion rate
- weekly active users

## Secondary Metrics
- roadmap completion
- squad activity
- deliverable submissions
